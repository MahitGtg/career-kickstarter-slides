Part 03 · Insert Flow

Insert — Three Phases

Every incoming order passes through validate → match → rest

① Validate

Guard against bad inputs before touching the book

· quantity > 0 · price > 0 · price aligns to tick_size

Fail → InsertFailure

→

② Match

Walk the opposing side while the price crosses

· BUY: sweep asks ≤ limit · SELL: sweep bids ≥ limit · Emit BookTrade per fill

Trades generated

→

③ Rest

Unfilled quantity joins the book as a resting order

· Create RestingOrder · Enqueue at price level (FIFO) · Update level quantities

remaining_qty may be 0

Returns InsertSuccess: order_id · timestamp · trades[] · traded_quantity · resting_quantity

Part 03 · Insert — Source

insert() — Full Method

order_book/book.py

def insert(self, params: InsertParams) -> InsertResult:
    if params.quantity <= 0:                                  # ①
        return InsertFailure("quantity must be positive")
    if params.price <= 0:
        return InsertFailure("price must be positive")
    if self._meta.tick_size > 0:
        ticks = round(params.price / self._meta.tick_size, 8)
        if abs(ticks - round(ticks)) > 1e-6:
            return InsertFailure(f"price {params.price} "
                f"is not a multiple of tick_size {self._meta.tick_size}")

    limit = _price_key(params.price)                         # ②
    incoming_id = self._alloc_order_id()
    remaining = params.quantity
    trades: list[BookTrade] = []

    if params.side == Side.BUY:                               # ③
        remaining = self._match_buy(incoming_id, limit, remaining, trades)
    else:
        remaining = self._match_sell(incoming_id, limit, remaining, trades)

    traded_qty = sum(t.quantity for t in trades)
    ts = self._now()

    if remaining > 0:                                         # ④
        resting = RestingOrder(
            order_id=incoming_id,
            order_book_id=self._meta.order_book_id,
            side=params.side, price=limit,
            remaining_quantity=remaining,
            username=params.username,
        )
        self._orders[incoming_id] = resting
        if params.side == Side.BUY:
            self._enqueue_bid(incoming_id, limit)
        else:
            self._enqueue_ask(incoming_id, limit)

    return InsertSuccess(                                     # ⑤
        order_id=incoming_id, timestamp=ts,
        trades=tuple(trades),
        traded_quantity=traded_qty,
        resting_quantity=remaining,
    )
① Validation guards — reject before any state mutation
② Round price to 4 d.p., allocate monotonic order_id
③ Branch on side → walk opposing book while price crosses
④ Unfilled remainder creates a RestingOrder, enqueued at its price level
⑤ Always returns success with trades, traded qty, and resting qty

Part 03 · Matching — Source

_match_buy() & _match_sell()

order_book/book.py — symmetric matching loops

_match_buy — aggressor is BUY

def _match_buy(self, incoming_id, limit, remaining, trades):
    while (remaining > 0
           and self._ask_prices
           and self._ask_prices[0] <= limit):
        level = self._ask_prices[0]
        q = self._ask_queues[level]
        passive_id = q[0]
        passive = self._orders[passive_id]
        qty = min(remaining, passive.remaining_quantity)
        trades.append(BookTrade(
            trade_id=self._alloc_trade_id(),
            order_book_id=self._meta.order_book_id,
            timestamp=self._now(),
            buy_order_id=incoming_id,
            sell_order_id=passive_id,
            price=level, quantity=qty,
            aggressor_side=Side.BUY,
        ))
        remaining -= qty
        self._reduce_resting(passive_id, qty, Side.SELL, level)
    return remaining

_match_sell — aggressor is SELL

def _match_sell(self, incoming_id, limit, remaining, trades):
    while remaining > 0 and self._bid_neg:
        level = -self._bid_neg[0]
        if level < limit:
            break
        q = self._bid_queues[level]
        passive_id = q[0]
        passive = self._orders[passive_id]
        qty = min(remaining, passive.remaining_quantity)
        trades.append(BookTrade(
            trade_id=self._alloc_trade_id(),
            order_book_id=self._meta.order_book_id,
            timestamp=self._now(),
            buy_order_id=passive_id,
            sell_order_id=incoming_id,
            price=level, quantity=qty,
            aggressor_side=Side.SELL,
        ))
        remaining -= qty
        self._reduce_resting(passive_id, qty, Side.BUY, level)
    return remaining
Loop guard:   BUY checks ask[0] <= limit · SELL checks -bid_neg[0] >= limit
Front of queue:   q[0] is always the time-priority passive order
Per fill:   _reduce_resting updates or removes the passive order

_enqueue_bid — add to bid side

def _enqueue_bid(self, order_id, price):
    if price not in self._bid_queues:
        self._bid_queues[price] = deque()
        bisect.insort(self._bid_neg, -price)
    self._bid_queues[price].append(order_id)
    self._apply_level_quantity_delta(
        Side.BUY, price,
        self._orders[order_id].remaining_quantity,
    )
New level?   Create deque, bisect.insort price into sorted list
Existing level?   Append to back of deque (FIFO)
Bid trick:   Store -price so ascending sort gives best bid first
_enqueue_ask is identical but stores price directly (ascending = best first)

Part 03 · Insert — During Matching

Reducing Resting Orders

Each fill decrements the passive order — fully filled orders are removed

_reduce_resting() source

def _reduce_resting(self, order_id, qty, side, price):
    resting = self._orders[order_id]
    new_rem = resting.remaining_quantity - qty
    if new_rem < 0:
        raise AssertionError("over-fill")

    self._apply_level_quantity_delta(side, price, -qty)

    if new_rem == 0:
        del self._orders[order_id]
        if side == Side.BUY:
            self._pop_front_bid(order_id, price)
        else:
            self._pop_front_ask(order_id, price)
    else:
        self._orders[order_id] = replace(
            resting, remaining_quantity=new_rem
        )
new_rem == 0   → delete order, pop front of deque, clean up empty level
new_rem > 0   → dataclasses.replace() with new remaining, stays at front

Part 03 · Cancel Flow

Cancel — Remove a Resting Order

Lookup by order_id · remove from queue · adjust quantities · clean up

1.

Lookup order_id in _orders dict

O(1) lookup — if missing, return CancelFailure("unknown order_id")

2.

Record remaining_quantity

Returned to caller so they know how much was still resting

3.

Remove from price level queue

BUY → _remove_bid_order · SELL → _remove_ask_order

4.

Delete from _orders dict

Order no longer exists — future cancel returns failure

_remove_bid_order — source

def _remove_bid_order(self, order_id, price, quantity):
    q = self._bid_queues[price]
    q.remove(order_id)
    self._apply_level_quantity_delta(
        Side.BUY, price, -quantity)
    self._clear_bid_level_if_empty(price)
Unlike matching:   can remove from any position in the deque, not just front
Level cleanup:   if deque empties, delete the price level and remove from sorted price list
Quantity tracking:   aggregate level qty decremented by the full remaining_quantity

Returns CancelSuccess: cancellation_timestamp · remaining_quantity

Part 03 · Cancel — Source

cancel() & Cleanup Helpers

order_book/book.py

cancel() — entry point

def cancel(self, order_id: int) -> CancelResult:
    resting = self._orders.get(order_id)
    if resting is None:
        return CancelFailure("unknown order_id")

    rem = resting.remaining_quantity
    ts = self._now()

    if resting.side == Side.BUY:
        self._remove_bid_order(
            order_id, resting.price, rem)
    else:
        self._remove_ask_order(
            order_id, resting.price, rem)

    del self._orders[order_id]
    return CancelSuccess(
        cancellation_timestamp=ts,
        remaining_quantity=rem)

_clear_bid_level_if_empty

def _clear_bid_level_if_empty(self, price):
    q = self._bid_queues.get(price)
    if q is None or q:   # still has orders
        return
    del self._bid_queues[price]
    neg = -price
    idx = bisect.bisect_left(self._bid_neg, neg)
    if (idx < len(self._bid_neg)
            and self._bid_neg[idx] == neg):
        self._bid_neg.pop(idx)

_apply_level_quantity_delta

def _apply_level_quantity_delta(
        self, side, price, quantity_delta):
    levels = (self._bid_level_quantities
              if side == Side.BUY
              else self._ask_level_quantities)
    new_qty = levels.get(price, 0) + quantity_delta
    if new_qty == 0:
        levels.pop(price, None)
        return
    levels[price] = new_qty

Part 03 · Internal Structures

Book Data Structures

From __init__ — sorted price lists · FIFO deques per level · aggregate quantities

def __init__(self, meta: OrderBookMeta, *,
             tick_clock=None):
    self._meta = meta
    self._tick_clock = tick_clock or (
        lambda: int(time.time() * 1000) & ((1 << 62) - 1))

    self._next_order_id = 1
    self._next_trade_id = 1

    self._orders: dict[int, RestingOrder] = {}

    # Bid side: store -price sorted ascending
    self._bid_neg: list[float] = []
    self._bid_queues: dict[float, deque[int]] = {}
    self._bid_level_quantities: dict[float, int] = {}

    # Ask side: store price sorted ascending
    self._ask_prices: list[float] = []
    self._ask_queues: dict[float, deque[int]] = {}
    self._ask_level_quantities: dict[float, int] = {}

_orders

dict[int, RestingOrder] — all live orders by order_id. O(1) lookup for cancel and matching.

Price levels

_bid_neg / _ask_prices — sorted lists. Bids store -price so ascending = best first. bisect.insort for O(log n) insert.

Level queues

dict[float, deque[int]] — FIFO queue of order_ids at each price. Front = highest time priority. Matching pops front; cancel can remove any position.

Level quantities

dict[float, int] — aggregate remaining_quantity per price level. Maintained incrementally; entry removed when zero.
